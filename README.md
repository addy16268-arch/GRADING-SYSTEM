echo "# Python Grading Function

This project contains a simple Python function that prints a grade based on the marks provided.

## Code

\`\`\`python
def grade(marks):
    if marks >= 90:
        print(\"A\")
    elif marks >= 80:
        print(\"B\")
    elif marks >= 70:
        print(\"C\")
    else:
        print(\"D\")
\`\`\`

## Explanation
- If \`marks >= 90\`, the program prints **\"A\"**.
- If \`80 <= marks < 90\`, the program prints **\"B\"**.
- If \`70 <= marks < 80\`, the program prints **\"C\"**.
- Otherwise (\`marks < 70\`), it prints **\"D\"**.

## Example Usage
\`\`\`python
grade(95)   # Output: A
grade(83)   # Output: B
grade(78)   # Output: C
grade(65)   # Output: D
\`\`\`

## How to Run
1. Copy the function into a Python file, e.g., \`grading.py\`.
2. Call the function with a mark value:
   \`\`\`python
   grade(85)
   \`\`\`
3. The corresponding grade will be printed in the terminal." > README.md
